เขตข้อมูลข้อความในฐานข้อมูลเดสก์ท็อป (.accdb) เรียนรู้เพิ่มเติมเกี่ยวกับชนิดข้อมูลสําหรับฐานข้อมูล Access บนเดสก์ท็อป เขตข้อมูลข้อความใน Access Web App ข้อความยาว ใน Access Web App เขตข้อมูล ข้อความยาว สามารถจัดเก็บได้สูงสุด 2^30-1 ไบต์ และเทียบเท่ากับชนิดข้อมูล SQL Server ของ nvarchar(max) ถ้าคุณต้องการ คุณสามารถตั้งค่าขีดจํากัดอักขระเพื่อป้องกันไม่ให้ผู้ใช้ของคุณใช้ความจุเต็มของเขตข้อมูลได้ คุณไม่สามารถจัดเก็บ Rich Text ใน Access Web App ได้
